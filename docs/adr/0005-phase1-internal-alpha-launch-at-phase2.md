# 0005 — Phase 1 is an internal alpha; public launch gated on Phase 2

**Status:** accepted

The Phase-1 RAW editor is built as an internal foundation and dogfooding alpha, **not**
a public launch. The public launch is gated on Phase 2 (local AI Recognition: skin/face
retouch + culling) — the first capability that beats Lightroom at something it cannot do.

**Why:** Phase 1 without AI competes head-on with a 15-year incumbent (Lightroom) on its
strongest ground — RAW development — using only incremental advantages (speed, offline,
price, Linux). Launching there risks branding the product as "a slower-to-arrive
Lightroom clone." The differentiators (AI batch retouch/culling) don't exist until
Phase 2, so that is the honest launch moment.

**Consequences:**
- Phase-1 exit criterion changes from "beat Lightroom on a 300-image export" to "solid,
  parity-correct foundation good enough to build AI on and to dogfood internally."
- No public revenue/feedback during Phase 1 — mitigated by a private alpha cohort.
- Reversible if a niche wedge (e.g. Linux-only beachhead) proves compelling earlier.
